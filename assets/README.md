# Assets Directory

This directory contains media files for the project documentation.

## Demo Video

The demo video is located here:
- `DP-Demo.mp4` (current demo video)

Supported formats:
- `.mp4` (recommended - best browser compatibility)
- `.webm` (alternative format)
- `.mov` (alternative format)

### Video Recommendations

- **Format**: MP4 (H.264 codec) for best compatibility
- **Resolution**: 1920x1080 (1080p) or 1280x720 (720p)
- **Duration**: 2-5 minutes recommended
- **File size**: Keep under 50MB if possible (GitHub has file size limits)

### How to Embed

The video will be automatically embedded in the main README.md. The current setup uses HTML5 video tags which work on GitHub.

If you prefer to host on YouTube or Vimeo instead:
1. Upload your video to YouTube/Vimeo
2. Update the README.md to use the embed code provided by the platform
